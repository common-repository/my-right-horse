== Description ==
Show your available horses easily on your Wordpress site. Requires a My Right Horse account and API Key. https://www.myrighthorse.org/

== Installation ==
1) Upload zip file or download from WordPress plugin's repository.
2) Activate plugin
3) Navigate to My Right Horse / Settings and enter your API key found under your My Right Horse account Embed & API Access tab https://www.myrighthorse.org/api_keys
4) Change the appearance including 2/3/4 columns, a detail view in a lightbox, contact information, style and filtering options.
5) Place the shortcode [horse_listing] wherever you would like the listings to appear, please note that you will need a wide column to display this information
